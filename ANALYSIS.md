# Анализ целевой аудитории (ЦА) для HH Vacancy Analyzer Bot

## 1. Методология
Анализ проведен на основе функциональности бота, сравнения с существующими решениями ("Типовые решения на рынке") и типовых потребностей пользователей, взаимодействующих с рынком труда.

## 2. Сегментация целевой аудитории
Целевую аудиторию бота можно разделить на три основных сегмента: Соискатели, HR-специалисты и рекрутеры, а также Аналитики рынка труда.

### Сегмент A: Соискатели
Это самая широкая аудитория, которую можно разбить на подгруппы.

*   **A1: Junior-специалисты и выпускники**
    *   **Проблема:** Слабое понимание рынка. Не знают, какие навыки (скиллы) действительно нужны, какую зарплату просить, в каких городах больше вакансий.
    *   **Потребности:** Узнать "входные требования" к профессии, получить список ключевых навыков для изучения, понять уровень зарплат на старте карьеры.
    *   **Как бот решает проблему:** Предоставляет статистику по требуемому опыту, топу навыков и зарплатам. Генерирует "профиль идеального кандидата", который служит дорожной картой для обучения.
    *   **Боли:** Боязнь провала на собеседовании, нерелевантные ожидания по зарплате.

*   **A2: Middle и Senior-специалисты (активный и пассивный поиск)**
    *   **Проблема:** Хотят оценить свой "вес" на рынке, понять, не продешевили ли они. Интересуются востребованностью своих навыков в конкретных нишах. Ищут не просто вакансию, а лучшие условия.
    *   **Потребности:** Глубокий анализ зарплат по рынку, сравнение своего опыта с требованиями из вакансий. Понимание, какие дополнительные навыки повысят стоимость.
    *   **Как бот решает проблему:** Анализ зарплат (среднее, медиана), анализ востребованности профессиональных ролей. Функция "Анализ резюме" напрямую сравнивает скиллы пользователя с "профилем идеального кандидата" и оценивает шансы.
    *   **Боли:** "Рыночная цена" часто непрозрачна, сложно объективно оценить свою конкурентоспособность.

*   **A3: Соискатели, ориентированные на релокацию / удаленную работу**
    *   **Проблема:** Поиск работы в другом регионе или стране СНГ. Незнание локальных зарплат и требований.
    *   **Потребности:** Возможность искать вакансии сразу на нескольких платформах (hh.kz, hh.uz, rabota.by) и получать статистику именно по интересующему рынку.
    *   **Как бот решает проблему:** Поддержка множества платформ HH и анализ топа городов по количеству вакансий. Конвертация валют при анализе зарплат.
    *   **Боли:** Сложность оценки рынка в другой стране без глубокого погружения.

### Сегмент B: HR-специалисты и рекрутеры

*   **B1: In-house рекрутеры и фриланс-рекрутеры**
    *   **Проблема:** Трата времени на ручной сбор информации для составления вакансионного профиля (Job Description). Нужно быстро понять, что сейчас предлагают конкуренты за похожих специалистов.
    *   **Потребности:** Быстрый срез рынка по конкретной позиции: зарплатная вилка, список ключевых требований, популярные места работы (топ городов для поиска).
    *   **Как бот решает проблему:** Мгновенно собирает статистику по вакансиям и выгружает ее в Excel с графиками для отчета нанимающему менеджеру. Генерирует "профиль идеального кандидата", который можно использовать как основу для вакансии.
    *   **Боли:** Субъективизм при составлении требований к вакансии, долгий сбор рыночных данных вручную.

### Сегмент C: Аналитики рынка труда и исследователи

*   **C1: Маркетологи, продуктовые аналитики, студенты экономических специальностей**
    *   **Проблема:** Необходимость сбора и обработки структурированных данных о спросе на профессии для трендов, отчетов или курсовых работ.
    *   **Потребности:** Получение "сырых" и обработанных данных по вакансиям с возможностью их дальнейшего анализа.
    *   **Как бот решает проблему:** Предоставляет Excel-файл с полными данными по вакансиям и уже построенными графиками. Логирование запросов в БД создает базу для долгосрочного анализа трендов.
    *   **Боли:** Отсутствие простого инструмента для парсинга и первичной обработки данных с HH.

## 4. Конкурентные преимущества бота для ЦА (Сравнительный анализ)
Сравнивая с описанными в брифе решениями, HH Vacancy Analyzer Bot занимает уникальную нишу "аналитического помощника".

| Критерий | HH Vacancy Analyzer Bot |
| :--- | :--- |
| **Основная функция** | Анализ рынка и сравнение резюме |
| **Глубина анализа** | Высокая (статистика, тренды) |
| **AI-функции** | Есть (профиль идеального кандидата, анализ резюме) |
| **Работа с резюме** | Оценка шансов и рекомендации |
| **Экспорт данных** | Есть (Excel с графиками) |
| **Мультиплатформенность** | Да (hh.ru, hh.kz, rabota.by и др.) |
| **Ценность для HR/аналитиков** | Высокая (сбор и аналитика рынка) |

> **Вывод для ЦА:** Бот не просто показывает вакансии, а объясняет рынок. Это делает его незаменимым инструментом для тех, кто хочет принимать решения, основанные на данных, а не на догадках.
