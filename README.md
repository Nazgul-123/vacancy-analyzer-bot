# 🤖 HH Vacancy Analyzer Bot

Telegram-бот для анализа рынка вакансий на основе данных с [hh.ru](https://hh.ru) и других платформ HeadHunter. Бот собирает вакансии по заданному описанию, проводит статистический анализ, формирует профиль идеального кандидата и анализирует резюме пользователя.

---

## ✨ Возможности

- 🔍 **Поиск вакансий** по текстовому описанию через API HH
- 🌍 **Поддержка нескольких платформ**: hh.ru, rabota.by, hh.kz, hh.uz, hh1.az, headhunter.ge, headhunter.kg
- 📊 **Статистический анализ** вакансий:
  - Топ городов по количеству вакансий
  - Распределение по графику работы и занятости
  - Требуемый опыт работы
  - Профессиональные роли и ключевые навыки
  - Анализ зарплат (минимум, максимум, среднее, медиана) с конвертацией валют
- 📗 **Экспорт в Excel** с графиками (pie chart, bar chart) по каждому параметру
- 🏅 **Профиль идеального кандидата** — генерируется на основе описаний вакансий с помощью ИИ
- 📋 **Анализ резюме** — оценка шансов на оффер с рекомендациями, сравнение с профилем идеального кандидата
- 💾 **Логирование запросов** в MySQL базу данных

## 💬 Как пользоваться

```
1. Напишите боту название или описание вакансии
   например: "Python разработчик" или "Data Scientist ML"

2. Выберите платформу для поиска (hh.ru, hh.kz и др.)

3. Получите:
   - Статистику по найденным вакансиям в Telegram
   - Excel-файл с данными и графиками
   - Профиль идеального кандидата от ИИ

4. Нажмите кнопку "Отправить резюме для анализа"
   и отправьте своё резюме текстом

5. Получите анализ резюме с оценкой шансов на оффер
```

## ⚠️ Ограничения

- HH API возвращает максимум **2000 вакансий** за один запрос (20 страниц × 100)
- Telegram ограничивает длину сообщения **4096 символами** — длинные сообщения разбиваются автоматически
- g4f использует бесплатные провайдеры без API-ключей — доступность может меняться
